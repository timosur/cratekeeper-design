# Product Roadmap

## Sections

### 1. Dashboard
Card grid of all active events with quick status, current step, and recent job activity at a glance.

### 2. Event Detail
The core working surface — a 12-step rail (fetch → enrich → classify → review → scan → match → analyze → tag → apply → build → sync) with per-step panels, review lanes, quality gates, and live SSE progress.

### 3. Settings
Auth re-link for Spotify and Tidal, Anthropic API key and model selector, filesystem roots, bearer token, and the editable genre bucket list.

### 4. Master Library
Global library stats and browse — what's accumulated on disk across all processed events, with per-genre counts and storage usage.

### 5. Audit Log
Filterable timeline of every state-changing action, queryable by target kind and target id, with override reasons surfaced inline.
