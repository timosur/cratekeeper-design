# Product Roadmap

## Sections

### 1. Events
Card grid of all active events with quick status, current step, and recent job activity at a glance. Drilling into a card opens the per-event working surface (the 11-step rail with fetch → enrich → classify → review → scan → match → analyze → classify-tags → apply → build event → build library, per-step panels, review lanes, quality gates, and live SSE progress).

### 2. Settings
Auth re-link for Spotify (wishlist source) and Tidal (listening / URL export), Anthropic API key and model selector, filesystem roots, bearer token, and the editable genre bucket list.

### 3. Master Library
Global library stats and browse — what's accumulated on disk across all processed events, with per-genre counts and storage usage.

### 4. Audit Log
Filterable timeline of every state-changing action, queryable by target kind and target id, with override reasons surfaced inline.
